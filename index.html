About Me

Fake Currency Note Identification Using Deep Convolutional Neural Networks

Under the mentorship of Dr. S.K. Pal at DRDO(Defence Research & Development Organization of India), New Delhi, we examined the feasibility of learning security features of the new Indian currency notes using a Deep Learning model. Created synthetic datasets of real and fake currency note images of ₹2000, ₹500 (~1600, ~7000 images respectively). Used a VGG-16 Convolutional Neural Network model pre-trained on ImageNet (~14 million images), and fine-tuned it with our small datasets. The model could classify images in test sets with 100% and 67% accuracy respectively. The observations were reported to the SmartTech-2017 conference.

Modelling sentences using Convolutional Neural Networks

Implemented a Dynamic Convolutional Neural Network architecture which achieves semantic modelling of sentences and can be used in Sentiment Analysis and Machine Translation. One epoch of training resulted in accuracy of 78%, against the well-trained model’s 87% on the same dataset as reported in the original research paper. Phase-II of project will focus on the possible improvements in this model, and application to relevant NLP problems.

Music Recommendation using Acoustic Analysis & Machine Learning

18 high level features including loudness, tempo were extracted with the Acoustic Analysis API (Echonest). Used Whitening followed by Neighbourhood Component Analysis (NCA) to apply a linear transform on the raw feature space, and the songs from various genres were embedded into a Euclidean metric space. k-nearest neighbours was used to classify/recommend songs.

word2vec-Theano

Implemented a simple version of Mikolov et al.’s (Google) word2vec, using python library Theano for GPU optimization during training. This Skip-Gram model used Negative Sampling to learn the word embeddings, with 5 negative samples for each positive query word-context word sample. The model was trained on text from the Harry Potter Series, using vocabulary size = 10000, embedding dimension = 64 and Nesterov Momentum for updates. 10 epochs of training resulted in well learned word embeddings.

Xerox Research Innovation Challenge 2016

The aim was to predict risk of death (mortality) in patients admitted to intensive care units (ICU) within hospitals, given the time-series data of 6 Vitals and 25 Lab Investigations as variables. Extracted relevant statistical features from these variables and used the AdaBoost boosting algorithm with decision trees as the weak learners. Achieved sensitivity = 0.352, specificity = 0.989 and accuracy = 0.945 on final evaluation.

Disabling MAC Learning per VLAN

Completed this project as a summer intern in Arista Networks. Added support for disabling MAC Learning per VLAN on two of the chips used in switches developed by Arista. This functionality is desired for providing pseudo-wire support for VXLAN (Virtual Extensible Lan). Wrote breath tests and a product test for the reactors and helpers created.

Data Analysis

Used three different classifiers, k-NN, Naïve Bayes and Decision Tree on the "Census Income" dataset (archived in UCI ML repository) for the task of predicting whether or not the income of a person exceeds $50K/year. Compared and evaluated the performance of the three classifiers based on various factors such as size of the dataset, types of attributes, etc.

My current research interest lies in studying Convolutional Neural Network Architectures. The question : "Is the selected CNN optimal for the dataset in terms of accuracy and model size?" is of great importance in deep learning research in the present time. I am working under the mentorhship of Dr. S.K. Pal (DRDO, New Delhi), with this question as our research focus. Here is some interesting and encouraging research in this area:
  • PerforatedCNNs: In this paper, they used the idea of Code Perforation, a source code optimization strategy. The idea in code perforation is to simply skip every other step in the loop, or every two steps, or whatever the program can get away with skipping without sacrificing too much accuracy. So, for the deep learning problem, they find indices in the image matrix for which convolutions will be applied, and for the rest of the indices, the nearest neighbor value(with exact output calculated using convolution) will be used. They call these set of indices with exact output "perforation masks". Through their experiments CNNs like AlexNet and VGG-16 could be accelerated by 2x to 4x with a small decrease in accuracy using perforation.

  • DropNeuron: In this paper, they improvised on the Regularization part. Instead of regularizing each single weight separately in all the layers, they applied a grouping effect on incoming and outgoing weights of a neuron during regularisation to drop the neuron completely from the network. This could achieve a compression rate of above 60% in case of their CNN experiment !

  • Dynamic DNNs: They have done some really interesting work in the research. They combined the conventional back-propagation with reinforcement learning to train a 'control' module which, depending on the input can choose to skip some part of the network and thus save the computation for that part. Their focus was to optimize accuracy-efficiency cut-offs and through a number of experiments they proved that their Dynamic DNN could achieve it. They implemented it with a well-known library Torch. Moreover, this DDNN is deterministic, in contrast to the other stochastic approached they reference in the paper which have been tried along this line of research.

  • CNNVis: They created a visual analytics system to help researchers in diagnosing and refining CNNs by offline analysing 'snapsots' of a CNN cuptured during training. Advanced algorithms - rectangle packing, matrix reordering, edge bundling, biclustering were used.

  • BoostCNN: In this paper, they used the boosting concept where several weak-learners result in a strong learner. They experimented with CNNs with 8 different input size variations as the weak learners. In future work, they say that they plan to apply BoostCNN to general large scale classification datasets and include a more diverse set of networks with different depths and configurations to find the best structure. This research requires very heavy computing resources however.
